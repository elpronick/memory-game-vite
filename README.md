# Memory Game (Vite + JavaScript)

Juego de memoria desarrollado con JavaScript vanilla y Vite.

## Características

- Tablero dinámico
- Detección de parejas
- Sistema de movimientos
- Persistencia con localStorage
- Reinicio de partida

## Tecnologías

- JavaScript
- Vite
- SCSS
- LocalStorage API

## Instalación

```bash
git clone https://github.com/tuusuario/memory-game-vite.git
cd memory-game-vite
npm install
npm run dev
